# Calculadora RNS PVC

Calculadora profesional de cotizaciones de cielos falsos de PVC para **RNS PVC** (Santa Cruz, Bolivia).

## Funciones
- Cálculo automático de materiales (placas, perfilería, consumibles).
- Selección automática del material óptimo (4m / 6m) y orientación.
- Optimización de cortes y empalmes (Unión H).
- Visualizador del techo a escala.
- Tabla editable de cantidades y precios.
- Exportación a PDF / impresión.

## Desarrollo
```bash
npm install
npm run dev      # servidor local
npm run build    # genera dist/ estático
```
